There are two scripts in "code" folder, "FAVefficacy_modeling_0408_24D.R" and "FAVefficacy_modeling_1418_glyphosate.R"

"FAVefficacy_modeling_0408_24D.R" fits a generalized additive mixed model (GAMM) to data from 2004-2008 time period only 
for sites that were treated with 2,4-D and not treated with glyphosate

"FAVefficacy_modeling_1418_glyphosate.R" similarly fits a generalized additive mixed model (GAMM) to data from 2014-2018 
time period only for sites that were treated with glyphosate and not treated with 2,4-D

There is one master dataset in the "data" folder which includes an integrated dataset for the entire study time period

Any figures created by the scripts will appear in the "figures" folder
